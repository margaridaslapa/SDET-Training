
# ⚓ SDET Training

This repository features my intensive 5-week training program focused on Software Engineering and QA Automation.

To ensure practical application, I have integrated these technical concepts into real-world projects for MariTech, my personal project dedicated to technological solutions for the maritime sector. Moving beyond generic exercises, I developed tools that solve logistics, telemetry, and port infrastructure challenges.


## 💻 Technology and Implementation

This monorepo focuses on building robust software capable of handling critical data and complex environments:

* **Backend & Logic:** C# (.NET 8) for core systems and high-performance APIs.
* **Automation:** Testing frameworks (NUnit, Selenium with POM) and utility scripting in Python.
* **Infrastructure:** System automation using PowerShell and Bash (Linux/Windows).
* **Engineering Principles:** Focus on memory management (Stack/Heap), SOLID principles, and CI/CD.
## 📍 Roadmap

The project is organized into weekly milestones, with each level adding a new layer of technical complexity:

#### [Week 1: C# OOP & Memory Management](./week-01-csharp-oop-memory-management)
* **Focus:** C# fundamentals and execution in a **Linux** environment.
* **Projects:** Fleet & Engine Core, Marina Dock Manager, and Port Traffic Controller (Async).
* **Tech:** Inheritance, Interfaces, LINQ, and Stack/Heap allocation analysis.

#### [Week 2: Test Automation & Resilience](./week-02-automation-nunit-selenium-pom)
* **Focus:** Automation frameworks and software robustness.
* **Projects:** Maritime Maintenance UI (POM), Vessel Connectivity Checker, and Ballast Water Logic Test.
* **Tech:** NUnit, Selenium (Page Object Model), Python (Retries), and State Logic Testing.

#### [Week 3: API Testing & Scripting](./week-03-api-testing-rest-python-scripting)
* **Focus:** Backend testing and language versatility.
* **Projects:** Sensor API Validator (C#), Multi-source Telemetry Converter (Python), and AIS Collision Parser (Python).
* **Tech:** HttpClient (C#), Requests (Python), and JSON Schema Validation.

#### [Week 4: DevOps & Infrastructure](./week-04-devops-cicd-powershell-linux)
* **Focus:** System automation and pipelines.
* **Projects:** Backup Integrity Watchdog, Linux Log Monitor, and Automated Environment Setup.
* **Tech:** PowerShell Scripting, Bash, Docker, and Azure DevOps Pipelines.

#### [Week 5: Web API, Security & Cloud](./week-05-webapi-integration-testing-cloud)
* **Focus:** System architecture and security.
* **Projects:** Maritime Incident API, Crew Access Security (JWT), and MariTech Cloud Architecture.
* **Tech:** ASP.NET Core, SOLID, Mocking (Moq), JWT, and Cloud Architecture Design (Azure).
